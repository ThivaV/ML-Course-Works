> Create conda environment with python 3.8

```python
conda create -n <env_name> python=3.8
```

<!-- Blockquote -->

> Show available conda environments

```python
conda env list
```

<!-- Blockquote -->

> Activate the created environment

```python
conda activate <env_name>
```
